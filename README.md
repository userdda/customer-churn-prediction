# 电商用户流失预测项目

## 项目背景
电商行业年流失率高达 20%，严重影响客户生命周期价值。本项目基于 Kaggle 的用户行为数据集（10万+样本），使用机器学习模型进行预测和优化。

## 技术路线
1. 数据清洗与特征工程（Pandas）
2. 建模与评估（scikit-learn）
   - Logistic Regression
   - Random Forest
3. 客户分群与流失预测
4. 策略模拟与优化

## 项目结果
- 模型准确率：92%
- 召回率提升：25%
- 模拟优化策略可降低流失率 15%
- 预计年节省成本：500万
